# MLFlow Tutorial

Este repositório contém um passo a passo que aborda os principais recursos do MLFlow e foi baseado em um [tutorial](https://mlflow.org/docs/latest/tutorials-and-examples/tutorial.html) da documentação oficial da ferramenta.

Os passo a passos abordados no repositório foram divididos por componentes do MLFlow:

1. [Tracking](./1.tracking/README.md): Exemplo de como o MLFlow Tracking pode ser utilizado no gerencimento de experimentos.
2. [Model Registry](./2.model-registry/README.md): Exemplo de como registrar um modelo e carrega-lo para predição.
